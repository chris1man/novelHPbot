# Простой Telegram Бот на Node.js

Этот бот готов к деплою на Timeweb Cloud.

## Локальный запуск

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Создайте файл `.env` на основе `.env.example` и добавьте свой токен бота:
   ```
   BOT_TOKEN=ваш_токен
   ```
3. Запустите бота:
   ```bash
   npm start
   ```

## Деплой на Timeweb Cloud

Для развертывания этого приложения в Timeweb Cloud (App Platform) используйте следующие настройки:

### 1. Настройка приложения
- **Тип приложения**: Node.js

### 2. Команда сборки
- **Команда**: `npm run build`

### 3. Зависимости (Команда установки)
- **Команда**: `npm install`

### 3. Команда запуска
- **Команда**: `npm start` (обычно определяется автоматически из package.json)

### 4. Директория сборки
- **Путь**: `/`

### 5. Переменные окружения (Environment Variables)
Необходимо добавить переменную с токеном вашего бота:

| Имя переменной | Значение |
|----------------|----------|
| `BOT_TOKEN`    | Ваш токен от @BotFather |

---

### Процесс загрузки:
1. Загрузите код в ваш репозиторий GitHub.
2. В панели Timeweb Cloud выберите "Apps" -> "Создать" -> "Node.js".
3. Подключите ваш репозиторий.
4. Укажите настройки, описанные выше.
5. Нажмите "Развернуть".
